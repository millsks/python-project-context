# Golden Examples

Reference implementations demonstrating best practices for common patterns. AI assistants can use these as templates when generating similar code.

## Directory Structure

- **repository/** - Repository pattern examples with SQLAlchemy 2.0
- **service/** - Service layer examples with dependency injection
- **api/** - FastAPI endpoint examples with validation
- **testing/** - Test examples with pytest fixtures and parametrize

## Usage

These examples demonstrate:
- Clean architecture patterns (repository, service, API layers)
- Type hints and MyPy compatibility
- SQLAlchemy 2.0 style queries
- Dependency injection patterns
- Comprehensive testing approaches
- Error handling and validation

Copy and adapt these patterns for your own code. AI assistants can reference these when you ask them to "follow the examples/ pattern."
